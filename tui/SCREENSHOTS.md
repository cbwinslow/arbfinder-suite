## Color Scheme

The actual TUI uses the following color scheme:

- **Title Bar**: Purple gradient (`#7D56F4`)
- **Active Tab**: White text on purple background
- **Inactive Tab**: Gray text on dark background
- **Input Fields**: Cyan borders with white text
- **Selected Items**: Purple highlight
- **Success Messages**: Green text
- **Error Messages**: Red text
- **Info Text**: Gray/dim text
- **Help Text**: Gray italic text

## Responsive Layout

The TUI adapts to different terminal sizes:

**Minimum**: 80 columns x 24 rows
**Recommended**: 100 columns x 30 rows
**Maximum**: Scales to any size

Content is dynamically sized based on available space, with:
- Responsive pane heights
- Automatic text truncation
- Adaptive pagination
- Scrollable content areas
