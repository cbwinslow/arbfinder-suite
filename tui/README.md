# ArbFinder TUI

The Bubbletea-based Terminal User Interface for ArbFinder Suite.

## Quick Start

```bash
# Build the TUI
make build-tui

# Run the TUI
make run-tui
```

Or directly:

```bash
cd tui
go run .
```

See the [full documentation](../docs/tui/README.md) for detailed information.
